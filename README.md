# negativ tools

Interní dashboard pro archviz workflow - ortho mapy, 3D terén, OSM budovy, AI generování a V-Ray integrace.

## Funkce

- **Ortho Map** - Stažení ortofoto s odstraněním stínů (ComfyUI)
- **3D Terrain** - Výškové mapy s ortho texturou
- **OSM Buildings** - Kontextové budovy z OpenStreetMap
- **AI Building Generator** - 3D modely z fotografií
- **AI Materials** - PBR textury a materiály
- **Creative Upscale** - AI upscale z V-Ray framebufferu
- **AI Atmosphere** - Atmosférické varianty renderů

## Instalace

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide Icons
